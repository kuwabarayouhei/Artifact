# 盗難車対策掲示板
近年問題になっている車の盗難についての情報を共有し、
twitterにて拡散することができるサイトです。


# 使用技術
- PHP 8.0.23  
- Laravel Framework 6.20.44  
- MariaDB  
- AWS  
    - cloud9  
    - S3  
- Twitter API  


# 機能一覧
- ユーザー登録機能,ログイン機能  
- 投稿機能  
    - 複数枚の画像投稿機能  
    - 投稿した内容をTwitterにて特定のアカウントから発信  
- コメント機能  
- ページネーション機能
